# Birse Thomas Design

Architecture and interior design studio website.

## Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19
- **Styling:** Tailwind CSS v4
- **Animation:** GSAP
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
src/
├── app/            # Next.js App Router pages and layout
├── components/     # Reusable UI components
└── lib/            # Shared constants and utilities
```

## Build

```bash
npm run build
npm start
```
